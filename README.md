# Coze 多平台内容同步 Skill

Coze (扣子) 技能商店技能 — 在微信公众号、小红书、抖音之间一键转换内容格式。

## 功能模块

### 公众号 → 小红书
- 深度文章 → 种草笔记
- 标题精简+emoji排版
- 段落拆分+互动引导+话题标签

### 小红书 → 抖音
- 图文笔记 → 短视频脚本
- 分镜设计+Hook优化
- 口播文案+音效建议

### 抖音 → 公众号
- 视频口播 → 结构化图文
- 背景知识补充+深度分析
- 配图位置标注+专业语气调整

## 使用方式

在Coze平台搜索「多平台内容同步助手」或通过以下输入：
```
/内容同步 [粘贴内容] 源平台:weixin 目标平台:xiaohongshu
```

## 目录结构
- `SKILL.md` — 主技能定义
- `prompts/` — 各方向转换规则
- `workflow/` — 同步工作流
- `templates/` — 各平台模板库
