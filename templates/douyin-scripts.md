# 抖音脚本模板

## 模板1：知识型
```
视频标题：XXX知识科普

【分镜1】0-3s
画面：博主直视镜头 + 惊讶表情
文案："你知道吗？XXX"
音效：悬疑音效

【分镜2】3-10s
画面：展示核心内容/数据/图表
文案："事实上，XXX（核心事实）"
字幕：突出关键数字

【分镜3】10-25s
画面：操作展示/场景演示
文案："具体来说，XXX（解决方案）"
音效：轻快BGM

【分镜4】25-40s
画面：博主总结 + 引导关注
文案："所以XXX，关注我，了解更多干货"
音效：结束音效

#知识科普 #干货
```

## 模板2：娱乐型
```
视频标题：XXX搞笑/有趣话题

【分镜1】0-3s
画面：反转/意外画面
文案："事情的经过是这样的..."
音效：悬疑→突然转换

【分镜2】3-15s
画面：事件A-事件B-事件C快速切换
文案：快节奏讲述故事
音效：快节奏BGM

【分镜3】15-30s
画面：高潮/反转
文案："结果你猜怎么着？XXX"
音效：爆笑音效

#搞笑 #日常
```

## 模板3：带货型
```
视频标题：XXX产品推荐

【分镜1】0-3s
画面：痛点展示
文案："你是不是也有这个问题？"
音效：叹气音效

【分镜2】3-15s
画面：产品亮相/功能展示
文案："直到我发现了这个XXX"
音效：惊喜音效

【分镜3】15-30s
画面：使用效果对比
文案："用了之后XXX（效果描述）"
字幕：价格/优惠信息

【分镜4】30-40s
画面：博主推荐 + 购买引导
文案："赶紧试试吧，链接在左下角"
音效：结束BGM

#好物推荐 #测评
```
