# 公众号文章结构模板

## 模板1：干货型
```
标题：XXX（含数字+价值承诺）

导语：XXX（全文核心观点，2-3句）

一、XXX（第一个核心观点）
（论据+案例+数据）
（小结）

二、XXX（第二个核心观点）
（论据+案例+数据）
（小结）

三、XXX（第三个核心观点）
（论据+案例+数据）
（小结）

总结
（核心观点回顾）
（行动建议）
（互动引导：你觉得呢？欢迎留言讨论）
```

## 模板2：故事型
```
标题：XXX（有故事感和悬念）

导语：XXX（故事背景简述）

一、开始
（故事的起点）
（遇到的问题和困境）

二、转折
（关键的发现或机遇）
（做出的改变）

三、高潮
（核心的经历和体验）
（最重要的感悟）

四、结局
（结果和收获）
（给读者的建议）

（升华主题的金句）
```
