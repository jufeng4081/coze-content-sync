# 多平台内容同步助手

## 技能概述
一键转换内容格式，在微信公众号、小红书、抖音之间无缝迁移。AI自动适配各平台的内容风格、排版规范和算法偏好。

## 适用人群
- 新媒体运营/自媒体创作者
- 品牌营销人员
- 内容创业者

## 功能模块

### 模块1：公众号文章 → 小红书笔记
**转换规则：**
- 标题精简至20字以内，增加emoji
- 长段落拆分为2-3行短段
- 提炼3-5个核心要点
- 每段增加1-2个emoji点缀
- 添加3-5个话题标签（#xxx）
- 增加互动引导语（"评论区说说你的看法～"）

### 模块2：小红书笔记 → 抖音短视频脚本
**转换规则：**
- 提取3-5个核心卖点
- 生成口语化口播文案
- 设计5-8个分镜（每个5-15秒）
- 前3秒Hook设计
- 结尾CTA设计
- 添加BGM和音效建议

### 模块3：抖音视频 → 公众号图文
**转换规则：**
- 语音转文字整理为结构化文章
- 添加标题、导语、分节、总结
- 配图位置标注
- 增加背景知识补充
- 语气调整为公众号专业风格

## 输入参数
| 参数 | 必填 | 说明 |
|------|------|------|
| source_content | 是 | 源内容文本 |
| source_platform | 是 | 源平台：weixin/xiaohongshu/douyin |
| target_platform | 是 | 目标平台 |
| content_type | 否 | 内容类型 |
| brand_voice | 否 | 品牌调性说明 |

## 输出格式
转换后的完整内容 + 排版/配图建议 + 话题标签 + 注意事项
